# CSV Ingestion using PySpark

This project reads a CSV file using PySpark, processes it, and saves the output in Parquet format.

## Features
- Read CSV file
- Infer schema
- Show data and schema
- Save as Parquet

## Project Structure
csv-ingestion-pyspark/
- main.py
- data/movies.csv
- .gitignore
- README.md

## How to Run
1. Install PySpark:
   pip install pyspark

2. Run the project:
   python main.py

## Output
- Displays sample data
- Prints schema
- Saves output in `output/data_parquet`
